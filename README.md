# JEP SDK for JavaScript

This package provides JavaScript/TypeScript tools for working with the JEP protocol.

## Relationship to JEP-ERC Bridge

This SDK is designed to work with the [jep-erc-bridge](https://github.com/jep-eth/jep-erc-bridge) contracts. It handles:
- Generating JEP Receipts
- Signing receipts with Ed25519
- Preparing data for on-chain verification

## Installation

```bash
npm install @jep-eth/sdk# jep-sdk-js
JavaScript SDK for JEP protocol - generate and verify JEP Receipts
